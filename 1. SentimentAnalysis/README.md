# Sentiment Analysis of IMDB Movie Reviews

This project performs **binary sentiment classification** (positive vs negative)
on IMDB movie reviews using classical NLP techniques and machine learning.

## Project Structure

- `notebooks/` → Jupyter notebook with all code
- `requirements.txt` → Python dependencies for this project

## Key Steps

1. Import required libraries
2. Load dataset
3. Preprocess text
4. Feature extraction (TF-IDF)
5. Train models (Logistic Regression, Naive Bayes, SVM)
6. Evaluate performance
7. Visualize results (word clouds, confusion matrix)
